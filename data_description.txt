Источник текстов:
Англо-русские тексты ООН с сайта https://conferences.unite.un.org/UNCorpus/

С сайта ООН я скачал комплект файлов, из которых важны три нижеуказанных файла. В каждом из них 23 239 280 строк (текст уже разбит на предложения, каждая строка — предложение):
	UNv1.0.en-ru.en - английский текст (3,45 GB)
	UNv1.0.en-ru.ru - русский текст (6,95 GB)
	UNv1.0.en-ru.ids - индекс (каждая строка индекса относится к соответствующей строке содержательных файлов)

Эти файлы я не выкладываю в репозиторий из-за из размера.


Файлы в репозитории в папке data/:

s_res_en, s_res_ru -- Резолюции Совета безопасности ОНН, 42 421 строка-предложение в каждом (извлечены с помощью файла-индекса)

en_ru_sort_uniq_remsim_jaro_2std_40, en_ru_sort_uniq_remsim_jaro_tok_2std_40 -- Файлы с АНГ-РУС данными после обработки (в каждом файле 14 561 строка-предложение, один файл "сырой", другой - токенизированный)

lex_preproc40_cleaned -- Глоссарий, полученный в результате word alignment, 56 460 строк (уникальных АНГ слов — 8419)

en_ru_heroku_1000 - Сокращенный АНГ-РУС текстовый файл с "сырым" текстом (994 строки-предложения) для размещения в интернете

heroku_glossary - Сокращенный АНГ-РУС глоссарий из 6827 строк (уникальных АНГ слов 2291) для обеспечения работы сайта в интернете
